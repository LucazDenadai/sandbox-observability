# Sandbox Observability

Repositório de estudos pessoais sobre **Observabilidade**, criado para reforçar na prática os conceitos que as aulas da pós-graduação não cobriram com profundidade suficiente.

O objetivo é comparar duas ferramentas de APM (Application Performance Monitoring) implementando os mesmos cenários em cada uma delas, para no final escolher qual usar no projeto do Tech Challenge da pós-graduação.

> Projeto de estudo pessoal. Sem preocupação com boas práticas de repositório além do necessário para organização própria.

---

## Ferramentas em estudo

- [Datadog](#datadog)
- [New Relic](#new-relic)

Cada ferramenta terá sua própria pasta com implementação e anotações:

```
/datadog
/new-relic
```

---

## Tópicos a explorar

Baseado na ementa de "Monitoramento e Acesso Avançado":

- O que é APM e seus benefícios
- Introdução ao APM (Datadog / New Relic)
- Logs
- Infraestrutura (Infra monitoring)
- Criando alertas
- Criando dashboards
- Contexto / mapeamento (Service Map)
